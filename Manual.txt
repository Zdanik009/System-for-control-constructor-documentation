Описание программы "System For Control Constructor Documentation"

Описание главного окна программы:

	В меню главного окна содержатся следующие разделы:
	- Раздел "System", содержащий подразделы, дублирующие кнопки на рабочем поле главного окна ("Add Detail" , "Add Document" , "Add Set of Documents" , "Save session");
	- Раздел "Information", содержащий подразделы: "Help" , "About Programm" , "About the Author".

	На рабочем поле главного окна содержатся следующие элементы:
	- Раскрывающийся список содержащий название деталей;
	- Нередактируемое поле вывода информации;
	- Кнопка добавления детали "Add Detail";
	- Ячейка подтверждения готовности;
	- Раскрывающийся список содержащий название документов;
	- Кнопка добавления документа "Add Document";
	- Кнопка добавленя "Add Set of Documents";
	- Полоса отражающая прогресс сбора документов;
	- Кнопка подтверждения готовности "Confirm".

Принцип работы элементов главного окна:

	ЭЛЕМЕНТЫ РАБОЧЕГО ПОЛЯ	

	- Раскрывающийся список содержащий название деталей отражает все существующие в программе элементы. При выборе одного из элементов: 
	- в поле вывода информации отражается название выбранной детали, её номер по списку, документы необходимые для её разработки и состояние этих документов (если документ A отсутствует
	в строке состояния выводится "There is NO A;"если же все документы собраны, выводится "You have all needed documents") 
	- в раскрывающемся списке содержащем название документов отражается все добавленные текущей разработке документы.
	- активизируются кнопки "Add Detail" , "Add Document" , "Add Set of Documents". Если все необходимые документы собраны, также активизируется кнопка подтверждения готовности 
	"Confirm".
	
	- Раскрывающийся список содержащий название документов отражает отражается все добавленные текущей разработке документы. При выборе одного из документов в ячейке подвеждения 
	готовности отразится состояния этого документа (галочка - документ добавлен текущей разработке; ячейка пуста - у текущей разработки отсутствует документ);

	- Ячейка подтверждения готовности отражает состояние текущего документа принадлежащий текущей разработке. Кликая по ячейке можно изменять состояние документа: кликнув по пустой ячейке 
	в ячейке появляется галочка, тем самым вы подтверждаете, что текущий документ добавлен текущей разработке; кликнув по ячейке с галочкой в ячейке пропадет галочка, тем самым вы 
	устанавливаете, что у текущей разработки отсутствует выбранный документ. Можно изменять состояние любого документа содержащегося в соседнем раскрывающемсч списке, состояние 
	документов будет немедленно отражаться в строке состояния поля вывода информации и полосе прогресса.

	- Кнопка добавления детали "Add Detail" открывает модальное диалоговое окно (см. Описание диалогового окна "Add Detail" , Принцип работы элементов диалогового окна "Add Detail"), 
	которое позволяет добавить новую уникальную разработку, прикрепик к ней любой из существующих наборов документов.

	- Кнопка добавления документа "Add Document" открывает модальное диалоговое окно (см. Описание диалогового окна "Add Document" , Принцип работы элементов диалогового окна 
	"Add Document"), которое позволяет добавить новый уникальный документ к текущей разработке, либо ко всем разработкам.

	- Кнопка добавления набора документов "Add Set of Documents" открывает модальное диалоговое окно (см. Описание диалогового окна "Add Set of Documents" , Принцип работы элементов 
	диалогового окна "Add Set of Documents"), которое позволяет добавить новый уникальный набор документов для текущей разработки или изменить уже существующий набор текущей разработки.

	- Кнопка добавления детали "Confirm" подтверждает готовность детали для разработки. Кнопка активизируется, если все документы для текущей разработки в состоянии готовности. Нажатие 
	на кнопку подтверждает отправку текущей детали на разработку, деталь удаляется из списка разработок, в раскрывающемся списке названия деталей автоматически выбирается следующая 
	деталь, соответственно обновляются: поле вывода информации, раскрывающийся список названий документов, полоса отраджающая прогресс сбора документов.

	РАЗДЕЛЫ МЕНЮ

	- Раздел "System" служит для проведения операций с разработками, наборами документов и документами по отдельности. Содержит подразделы, дублирующие кнопки на рабочем поле главного
	окна: "Add Detail" , "Add Document" , "Add Set of Documents". (см. Описание диалогового окна "Add Detail" , Принцип работы элементов диалогового окна "Add Detail" , Описание 
	диалогового окна "Add Document" , Принцип работы элементов диалогового окна "Add Document" , Описание диалогового окна "Add Set of Documents" , Принцип работы элементов 
	диалогового окна "Add Set of Documents") а также "Save session" - позволяет сохранить историю всех операций совершенных за сессию в программе. История сохраняется в текстовом файле,
	пользователь имеет возможность выбрать место сохранения файла.

	- Раздел "Information" включает в себе инструкцию к использованию, информацию о программе и авторе. Содержит подразделы: "Help" - инструкция пользования программой;
	"About Programm" - информация о текущей версии программы и её компонентах; "About the Author" - информация об авторе программы.
	

Описание диалогового окна "Add Detail":
	
	На рабочем поле диалогового окна содержатся следующие элементы:
	- Поле ввода названия детали;
	- Раскрывающийся список содержащий название наборов документов;
	- Кнопка подтверждения создания и добавления новой разаботки "Add";
	- Кнопка закрытия диалогового окна "Cancel".

Принцип работы элементов диалогового окна "Add Detail":

	Поле ввода названия детали предназначено для ввода названия новой детали. Для создания нового имени необходимо ввести уникальное название, состоящее из прописных букв латинского
	алфавита (любые другие знаки запрещены). Для добавления детали с этим именем необходимо выбрать набор документов в раскрывающемся списке.

	Раскрывающийся список предназначен для выбора набора документа для создаваемой детали. Необходимо раскрыть список и выбрать любой из предложенных наборов. Выбор можно изменить в
	любой момент создания новой детали, после потверждения создания детали изменить набор документов можно будет только с помощью сервиса "Add Set of Documents". Без набора документов
	деталь создать невозможно. Как только набор выбран активизируется кнопка "Add".

	Кнопка "Add" предназначена для потверждения создания и добавления новой разработки. При нажатии на кнопку диалоговое окно закрывается; новая деталь создается в памяти программы и
	заносится раскрывающийся список названия деталей, автоматически становится текущей; обновляется информация в поле вывода информации, в списке документов принадлежащиъ текущей 
	разработке, в полосе отражающей прогресс сбора документов.

 	Кнопка "Cancel" предназначена для закрытия диалогового окна.


Описание диалогового окна "Add Doc":
	
	На рабочем поле диалогового окна содержатся следующие элементы:
	- Поле ввода названия документа;
	- Ячейка подтверждения;
	- Кнопка подтверждения создания и добавления новой документа "Add";
	- Кнопка закрытия диалогового окна "Cancel".

Принцип работы элементов диалогового окна "Add Doc":

	Поле ввода названия документа предназначено для ввода названия нового документа. Для создания нового документа необходимо ввести уникальное название, состоящее из одной заглавной
	буквы латинского алфавита (любые другие знаки запрещены).

	Ячейка подтверждения предназначен для подтверждения добавления созданого документа ко всем разработкам в списке деталей. Выбор можно изменить в любой момент создания новой документа,
	после потверждения создания документа изменить набор документов можно будет только с помощью сервиса "Add Set of Documents". 

	Кнопка "Add" предназначена для потверждения создания и добавления нового документа. При нажатии на кнопку диалоговое окно закрывается; новая документ создается в памяти программы и
	заносится в раскрывающийся список названия документов (текущей детали, либо всех деталей в зависимости от состояния ячейки подтверждения); обновляется информация в поле вывода 
	информации, в списке документов принадлежащиъ текущей разработке, в полосе отражающей прогресс сбора документов.

 	Кнопка "Cancel" предназначена для закрытия диалогового окна.
		

Описание диалогового окна "Add Set of Documents":
	
	На рабочем поле диалогового окна содержатся следующие элементы:
	- Поле ввода названия набора документов;
	- Поле вывода информации;
	- Ячейка подтверждения готовности;
	- Раскрывающийся список содержащий название документов;
	- Кнопка подтверждения создания и добавления новой разаботки "Add";
	- Кнопка закрытия диалогового окна "Cancel".

Принцип работы элементов диалогового окна "Add Set of Documents":

	Поле ввода названия набора документов предназначено для ввода названия нового набора документов (изначально в поле будет находится название текущей детали). Для создания нового 
	набора необходимо ввести уникальное название, состоящее из прописных букв латинскогоалфавита (любые другие знаки запрещены). Название набора отразится в поле вывода информации.

	Поле вывода информации предназначено для отражения текущего состояния создаваемого набора. Содержит в себе название набора документов и названия документов добавленных в 
	разрабатываемый набор. При изменении названия в поле ввода, добавлении или удалении документов из набора поле вывода информации автоматически обновляется.

	Раскрывающийся список предназначен для выбора документов для создаваемого набора документов. Раскрывающийся список содержит все сущетсвующие в программе документы. Необходимо 
	раскрыть список и выбрать любой из предложенных документов, ячейка подтвеждения готовности отразит принадлежность выбранного документа, и кликнуть пл ячейке подтверждения готовности
	для добавления или удаления документа из набора. Выбор можно изменить в любой момент создания нового набора документов. Набор без документов создать невозможно. Кнопка "Add" 
	активизируется если в наборе содержится хотя бы один документ.

	Ячейка подтверждения готовности предназначена для отражения принадлежности документа к создаваемому набору, а также для добавления или удаления документа из создаваемого набора.
	Для добавления документа в набор необходимо, выбрать документ в раскрывающемся списке и активировать ячейку подтверждения готовности, для удаления ячейку нужно деактивировать.
	Информация о добавлении или удалении документа автоматически отражается в поле вывода информации 

	Кнопка "Add" предназначена для потверждения создания и добавления нового набора. При нажатии на кнопку диалоговое окно закрывается; новый набор создается в памяти программы и
	присваивается текущей разработке; обновляется информация в поле вывода информации, в списке документов принадлежащих текущей разработке, в полосе отражающей прогресс сбора 
	документов.

 	Кнопка "Cancel" предназначена для закрытия диалогового окна.
	
Описание диалогового окна "Save session": 
	
	В качестве диалогового окна "Save session" используется стандартное диалоговое окно сохранения файлов. (см. Microsoft.DialogWindowForSaveFile)

Базовые инструкции пользования программой:
	После загрузки программы на старте для пользователя активен только раскрывающийся список с названиями деталей и разделы меню. Необходимо выбрать любую из предложенных в списке 
	деталей, после этого станут активными остальные элементы окна. В окне вывода информации вы можете увидеть всю информацию о выбранной детали и её наборе документов. Изменять состояние
	документов можно с помощью раскрывающегося списка с документами и ячейки подтверждения. Также вы можете добавлять новые детали, документы и даже целые наборы документов с помощью 
	кнопок соответственно "Add Detail" , "Add Document" , "Add Set of Document". Кнопка "Confirm" подтверждает готовность детали для разработки.
